Algumas Tecnologias associadas ao Desenvolvimento Web
HTML HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para estruturar páginas da Web.
CSS CSS (Cascading Style Sheets) é um mecanismo para aplicar estilos (cores, fontes, espaçamento, etc.) a uma página web, podendo ser aplicado diretamente nas tags HTML ou por meio de arquivos externos.
JavaScript JavaScript é uma linguagem de programação usada para criar interações dinâmicas em páginas web. Juntamente com HTML e CSS, é uma das três principais tecnologias da Web.
PHP PHP é uma linguagem de programação usada principalmente para desenvolvimento de aplicações do lado do servidor, permitindo a criação de conteúdos dinâmicos na Web.
